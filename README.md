# Arch
A portable archive manager for storing Z-compressed files

See the comments at the top of arch.adb for usage information

Arch requires [Z_Compression](https://github.com/jrcarter/Z_Compression) and the [PragmAda Reusable Components](https://github.com/jrcarter/PragmARC)

Arch is completely portable Ada 12; it has been tested with GNAT and ObjectAda on Linux and Windows
